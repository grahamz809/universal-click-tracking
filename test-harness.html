💡 Usage:
  1. Click the bookmark on any page (OHIO.edu, OSU, MiamiOH, etc.)
  2. A floating panel appears showing match counts by event
  3. Hover any element to see which event would fire
  4. Click any element to see its proposed parameters
  5. Console also logs full JSON of each inspected element

⚙️ How The Matching Works

The tool runs 16 pattern-matching rules (in priority order) on every element you hover. First match wins.

Priority Event Family Match Strategy
1generate_leadConversionform tag, form_button_submit, [data-form-type]
2event_rsvpConversionText match: "RSVP", "Register", "Confirm"
3file_downloadConversion[download] attr or .pdf/.docx/.xlsx URL
4cta_buttonEngagementCSS class: "button", "btn", .button.green
5cta_linkEngagementText: "Learn More", "Read More", jump links
6global_navEngagementInside <nav>, <footer>, breadcrumbs
7contact_clickEngagementtel:, mailto:, phone/email pattern
8web_elementEngagementAccordion, tabs, carousel, expand/collapse
9internal_searchDiscoveryform[action*=search], search input/button
10custom_filter_searchDiscovery[data-module], .program-finder, .filter
11news_contentContent & MediaInside article, .news, .story, "View all stories"
12videoContent & Mediaiframe youtube/vimeo, "Play video" text
13tool_interactionUtility & Support[data-tool], .calculator, .interactive-tool
14chatUtility & Support[class*="chat"], [aria-label*="chat"]
15exit_linkUtility & SupportExternal hostname (not current domain)
16404Utility & SupportPage title/status (page-level, not element)

🧪 How To Test

Method 1: Bookmarklet

  1. Drag the "🎯 Inspect Tracking" button above to your bookmarks bar
  2. Visit any page (OHIO.edu, OSU, MiamiOH, etc.)
  3. Click the bookmarklet
  4. Hover elements → see event match in the panel
  5. Click elements → see full proposed parameters

Method 2: Test on this page

  1. Install the bookmarklet above
  2. Click it while on this page
  3. Hover every demo element below
  4. Verify the correct event fires for each
  5. Check that match counts make sense

Method 3: Test on OSU / MiamiOH

The bookmarklet works on any website. Visit osu.edu or miamioh.edu, click the bookmarklet, and verify the mapping documented in PORTABILITY.md.
