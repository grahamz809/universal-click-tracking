Can't drag? Copy-paste this into a new bookmark's URL field:

💡 How to use it:
  1. Click the bookmark on any page (OHIO.edu, OSU, MiamiOH, etc.)
  2. A floating panel appears showing match counts by event
  3. Hover any element → see its event name, family, CSS selector, and 3 standard parameters
  4. Click any element → it's added to your framework list
  5. Click CSV → opens an export dialog to copy data into Excel or download as a .csv file
  6. Click Framework → shows your captured elements grouped by family
  7. Use – to minimize the panel, or ✕ to close it

📌 Alternative: Framework Inspector (Quick Visual Scan)

A lighter-weight tool that scans the page in one pass and draws colored outlines around every classified element. Hover any outlined element to see its event name and location in a tooltip.

Uses a simplified rule set (8 event types) — results may differ slightly from the main Click Tracker Framework.

⚙️ How the Matching Works

The tool runs taxonomy rules (priority order) plus a catch-all at the end that ensures every link and interactive element on the page is captured. Every match always includes the 3 universal parameters + CSS selector.

Universal Parameters (every event)

ParameterSource
link_click_urlel.href (links) or form.action (forms) or empty
web_element_locationPage region detection: breadcrumb, aux-menu, main-nav, hero, footer, body
click_textel.textContent trimmed (max 150 chars)
Event Family Match Strategy
generate_leadConversionform tag, form_button_submit, [data-form-type]
event_rsvpConversionText match: "RSVP", "Register", "Confirm"
file_downloadConversion[download] attr or .pdf/.docx/.xlsx URL
cta_linkEngagementButton-class links, "Learn More", card links, arrows, body text links
global_navEngagementInside main-nav, aux-menu, breadcrumb, footer elements
contact_clickEngagementtel:, mailto:, phone/email pattern
web_elementEngagementAccordion, tabs, carousel, expand/collapse, info indicators, modals
internal_searchDiscoveryform[action*=search], search input/button, funnelback block
custom_filter_searchDiscovery[data-module], .program-finder, .filter
news_contentContent & MediaInside article, .news, .story, "View all stories"
videoContent & Mediaiframe youtube/vimeo, "Play video" text, hero video
tool_interactionUtility & Support[data-tool], .calculator, .interactive-tool
chatUtility & Support[class*="chat"], [aria-label*="chat"]
exit_linkUtility & SupportExternal hostname (not ohio.edu)
404Utility & SupportPage title/status (page-level, not element)
⏣ catch-allEngagementEvery unmatched link → cta_link, every unmatched button → web_element
